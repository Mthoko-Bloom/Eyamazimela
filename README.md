# Eyamazimela Pty Ltd — Website

Landing page for Eyamazimela Pty Ltd: premium event catering, elegant venue décor,
custom baking & celebration cakes, and professional media — Hammarsdale, KwaZulu-Natal.

> "We strive in making your special day perfect."

## Stack
- Next.js 14 (App Router, static export)
- Tailwind CSS
- Lucide icons

## Develop
```bash
npm install
npm run dev
```

## Deploy
Pushes to `main` auto-deploy to GitHub Pages via `.github/workflows/deploy-pages.yml`.
Also import-ready for Vercel (zero config).
